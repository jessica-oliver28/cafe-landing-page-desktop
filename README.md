# ☕ Café Aroma - Landing Page Desktop

## 📌 Sobre o projeto

**Café Aroma** é uma landing page que criei para uma cafeteria fictícia. O objetivo foi desenvolver uma página estática, com layout fixo para desktop, aplicando conceitos fundamentais de **HTML**, **CSS** e **JavaScript** que aprendi nos meus estudos de front-end.

Este projeto faz parte do meu portfólio e representa meu primeiro passo na construção de interfaces reais, com organização de código, preocupação com usabilidade e validação de formulário.

# 👩‍💻 O que eu fiz e o que aprendi

###  Estrutura HTML semântica
- Construí seções bem definidas: cabeçalho com navegação, banner principal, cardápio, serviços, formulário de contato e rodapé.
- Usei âncoras (`#id`) para que o clique no menu role suavemente até a seção desejada.
- Aprendi a importância de nomear classes e IDs de forma clara para facilitar a estilização e o JavaScript.

###  Estilização com CSS
- Defini um layout fixo de **1200px** (`.wrapper`) – escolha intencional para manter o foco no desktop.
- Utilizei **Flexbox** para alinhar os elementos do menu, serviços e contato, garantindo alinhamento e espaçamento uniformes.
- Criei efeitos visuais como `hover` nos botões e cards, gradiente sobre o banner e sombras suaves.
- Aprendi a centralizar elementos com `margin: 0 auto` e a controlar o tamanho das imagens com `object-fit: cover`.

###  JavaScript para interação
- Implementei validação do formulário de contato (campos obrigatórios e formato de e-mail).
- Usei `addEventListener` para capturar o envio, exibir mensagens de feedback e limpar o formulário após sucesso.
- Aprendi a importância de ler os valores **dentro** do evento `submit` para capturar o que o usuário digitou no momento.
- Adicionei um `setTimeout` para que a mensagem de feedback desapareça após 4 segundos, melhorando a experiência.

###  Gerenciamento de imagens
- Organizei todas as imagens em uma pasta `imagens/`, tornando o projeto limpo e fácil de manter.
- Usei imagens de exemplo (banner, expresso, cappuccino, croissant, latte) que podem ser substituídas facilmente.
